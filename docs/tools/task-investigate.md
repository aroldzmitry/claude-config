# /task:investigate

Explore a topic from multiple angles with web research and structured insights.

## Usage

```
/task:investigate how to implement real-time sync
/task:investigate
```

## What It Does

1. Parses topic, identifies domain and constraints
2. Asks 2-3 discovery questions (team size, current state, goals)
3. Analyzes from multiple perspectives (problem, stakeholders, risks, alternatives)
4. Performs web research tailored to context
5. Generates insights: key finding, reframe, quick wins, deep work
6. Offers to save at `~/.claude/investigations/{date}-{slug}.md`

## When to Use

- Exploring unfamiliar problem space
- Need research before implementation decisions
- Want structured analysis of trade-offs

## Model

opus (deep analysis, extended thinking)
